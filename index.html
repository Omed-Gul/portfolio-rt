Portfolio – BUT Réseaux & Télécommunications

Omed Gul Amini – 1ère année R&T

Objectif du portfolio

Ce portfolio présente des preuves issues de ma première année de BUT Réseaux & Télécommunications. L’objectif est de montrer ma progression dans les trois compétences principales : administrer les réseaux, connecter les systèmes et programmer des outils informatiques.

Compétence 1 – Administrer les réseaux et l’Internet

Niveau visé : Assister l’administrateur du réseau.

Pour cette compétence, j’utilise principalement la SAE12 sur l’architecture de mon réseau local. Ce travail m’a permis de représenter un réseau domestique, d’identifier les équipements, d’expliquer le rôle du DHCP, du DNS, du NAT, du Wi-Fi, du câblage et de l’adressage IP.

Preuves principales :

SAE12 Topologie réseau Plan d’adressage IP DHCP / DNS / NAT

Compétence 2 – Connecter les entreprises et les usagers

Niveau visé : Découvrir les transmissions et la ToIP.

Pour cette compétence, j’utilise la SAE22 sur la mesure et la caractérisation d’un signal. Le projet repose sur MATLAB, l’analyse fréquentielle, la FFT, le filtrage passe-bas et la synthèse sonore avec l’algorithme de Karplus-Strong. J’ajoute aussi le TP R204 sur la VoIP pour montrer le lien avec la téléphonie IP.

Preuves principales :

SAE22 MATLAB FFT Filtrage Karplus-Strong R204 VoIP

Voir les preuves Connecter

Compétence 3 – Créer des outils et applications informatiques pour les R&T

Niveau visé : S’intégrer dans un service informatique.

Pour cette compétence, j’utilise principalement la SAE23 sur l’application Vendée Globe. Ce projet m’a permis de travailler sur une base de données PostgreSQL, des requêtes SQL, une API REST avec Java/Tomcat, des réponses JSON, et une interface web en HTML, CSS, JavaScript et Bootstrap.

J’ajoute aussi des preuves issues de la SAE13, où j’ai manipulé des données audio et image avec MATLAB. Cela montre ma progression dans la lecture, la modification et la transformation de données par programme.

Preuves principales :

SAE23 PostgreSQL SQL Java / Tomcat API REST JSON HTML / CSS / JS SAE13 MATLAB

Bilan personnel

Cette première année m’a permis de découvrir plusieurs dimensions des Réseaux & Télécommunications : l’architecture réseau, les transmissions, la téléphonie IP, les systèmes, les bases de données et le développement web. Mon objectif est maintenant de renforcer mes bases techniques pour devenir plus autonome dans l’administration réseau et le développement d’outils informatiques.
