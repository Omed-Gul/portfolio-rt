Cette partie regroupe les preuves liées à la compétence **Programmer**.

Preuves principales :
- SAE23 : application web Vendée Globe

- Réponses JSON
- Frontend HTML, CSS, JavaScript et Bootstrap
- SAE13 : manipulation de données audio et image avec MATLAB
