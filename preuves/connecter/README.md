# Connecter les entreprises et les usagers

Cette partie regroupe les preuves liées à la compétence **Connecter**.

Preuves principales :
- SAE22 : mesure et caractérisation d’un signal avec MATLAB
- Synthèse sonore avec l’algorithme de Karplus-Strong
- Analyse fréquentielle avec FFT
- Filtrage passe-bas
- R204 : VoIP, téléphonie IP, CUCM, CUC, téléphones Cisco, VLAN voix
