Preuve principale : SAE22 – Mesurer et caractériser un signal

```

Dans cette SAE, j’ai travaillé sur la synthèse sonore avec MATLAB à partir de l’algorithme de Karplus-Strong. L’objectif était de générer un son de guitare numérique, d’analyser le signal obtenu, puis de le modifier avec un effet.

J’ai utilisé la génération de bruit blanc, une ligne à retard, un filtre passe-bas, la FFT pour observer le spectre, puis une saturation pour créer un effet de distorsion.

SAE22 MATLAB FFT Filtrage passe-bas Karplus-Strong Signal sonore Spectre fréquentiel
```

Preuve complémentaire : R204 – Téléphonie IP / VoIP

Le TP R204 m’a permis de découvrir une infrastructure de téléphonie IP avec Cisco CallManager, Cisco Unity Connection et des postes téléphoniques Cisco.

Ce que j’ai appris

```

J’ai compris qu’un signal peut être étudié dans le domaine temporel et dans le domaine fréquentiel. La FFT permet de visualiser les harmoniques, et le filtrage modifie le contenu fréquentiel.

Avec la VoIP, j’ai aussi compris que connecter des usagers ne dépend pas seulement du téléphone : il faut aussi des serveurs, une configuration réseau, du DHCP, des VLAN et une signalisation adaptée.

```
