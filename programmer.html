Preuve principale : SAE23 – Application Vendée Globe

La SAE23 est ma preuve principale pour la compétence Programmer. Le projet consistait à construire une application complète autour des données du Vendée Globe.

Le travail était organisé en plusieurs parties : base de données PostgreSQL, requêtes SQL, développement Java, API REST avec Tomcat, échange de données en JSON, puis interface web avec HTML, CSS, JavaScript et Bootstrap.

SAE23 PostgreSQL SQL Java Tomcat API REST JSON JavaScript Bootstrap

Preuve complémentaire : SAE13 – Traitement audio et image avec MATLAB

La SAE13 complète cette compétence, car elle montre mon utilisation d’outils informatiques pour traiter des données numériques.

Dans l’activité audio, j’ai manipulé un fichier sonore avec MATLAB : lecture, affichage du signal, découpage de phonèmes, reconstruction d’un mot et export audio. Dans l’activité image, j’ai travaillé sur les pixels, les couleurs, les zones d’image, le découpage et la reconstruction d’une image.

Ce que j’ai appris

J’ai compris qu’une application informatique n’est pas seulement une page web. Elle repose sur plusieurs couches : une base de données, un serveur, une API, un format d’échange de données et une interface utilisateur.

J’ai aussi compris l’importance de structurer son travail, documenter les fichiers, tester les différentes parties et relier le code aux besoins de l’utilisateur.
